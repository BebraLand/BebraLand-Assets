# BebraLand Media Kit

Welcome to the BebraLand Media Kit! This repository contains all the official public assets, logos, and graphics for the BebraLand community.

## Structure

The assets are organized into the following categories:

### 📁 `Public/Logos`
Contains the official text logos and branding graphics, primarily designed in our signature blocky/Minecraft style.
- `bebraland_application_2k.png` - "BEBRALAND APPLICATION" graphic.
- `bebraland_create_2.0_title_hd.png` - Main "BEBRALAND CREATE 2.0" logo.
- `bebraland_rules_2k.png` - "BEBRALAND RULES" graphic.
- `bebraland_create_2_0_alt.png` - Alternative Create 2.0 logo.

### 📁 `Public/Icons`
Contains various icon assets, including our beaver mascot and website favicons.
- Mascot images in various resolutions (e.g., `beaver1920x1920.png`, `beaver256x256.png`).
- `favicons/` - Contains standard and old `.ico` and `.png` favicons for web use.

### 📁 `Public/Backgrounds`
Contains background images, landscapes, and screenshot-style assets for banners or general use.
- General landscape/biome images (e.g., `terralith biome.jpg`).
- UI/screen backgrounds (e.g., `smartscreen.png`).

## Usage Guidelines
These assets are provided for the Bebraland community to use in videos, streams, fan art, or related projects. Please use them responsibly and maintain the cozy, friendly vibe of the community!
